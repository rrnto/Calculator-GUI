# Multi-Operation Calculator (Python & JavaScript)

This repository contains two implementations of a simple calculator that supports addition, subtraction, multiplication, and division:

## Python Version
- Calculator for quick arithmetic
- Optional GUI version using Tkinter
- Handles invalid input and division by zero

## JavaScript Version
- Web-based calculator using HTML, CSS, and JS
- Icon-style dropdown for selecting operations (+, −, ×, ÷)
- Dark-themed, modern UI
- Fully client-side, works in any browser

## Usage

### Python
Run the Python script:
```bash
python main.py
